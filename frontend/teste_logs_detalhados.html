📊 Gráfico de Teste

🔍 Console Logs (Abra o DevTools):

Abra o DevTools (F12) e vá para a aba Console para ver os logs detalhados.

Os logs mostrarão exatamente o que está acontecendo com os dados.

📋 O que procurar nos logs:
  1. 🔍 === INÍCIO renderSalesChart === - Início da função
  2. 📊 Dados recebidos: - O que está sendo passado
  3. 📊 Tipo dos dados: - Array, objeto, null, etc.
  4. 📊 Tamanho dos dados: - Quantos registros
  5. 📦 Processando venda X: - Cada venda individual
  6. 📅 Data, Ano, Mês, Valor: - Campos extraídos
  7. ✅ Adicionado ao ano: - Distribuição correta
  8. 📊 === DADOS PROCESSADOS === - Resultado final
🚨 Possíveis Problemas:
  • Dados vazios: "Tamanho dos dados: 0"
  • Formato incorreto: "Tipo dos dados: object" mas não array
  • Campos faltando: "Data: Invalid Date"
  • Valores zero: "Valor: 0" para todas as vendas
  • Anos incorretos: Vendas em anos que não são atual/anterior
